# Readme

Sekcja poświęcona architekturze sieci oraz komunikacji między urządzeniami i maszynami w środowisku laboratoryjnym.

# Zakres

- konfiguracja sieci lokalnej i komunikacji między urządzeniami,
- organizacja oraz segmentacja ruchu sieciowego,
- konfiguracja dostępu zdalnego i połączeń między hostami,
- analiza oraz monitoring ruchu sieciowego,
- testowanie stabilności i bezpieczeństwa komunikacji,
- przygotowanie infrastruktury pod narzędzia SOC oraz analizę bezpieczeństwa.

# Cel

Celem jest zbudowanie stabilnego i bezpiecznego środowiska sieciowego umożliwiającego:
- monitorowanie ruchu,
- imitację oraz analizę zagrożeń,
- integrację narzędzi bezpieczeństwa,
